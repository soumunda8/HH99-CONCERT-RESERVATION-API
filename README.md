# 콘서트 예약 서비스

본 프로젝트는 콘서트 예약 서비스로, 사용자가 콘서트를 예약할 수 있는 API 기능을 제공하는 프로젝트입니다.

- **개발 환경 설정**
    - **기본 패키지 구조** : Clean Layered Architecture
    - **기술 스택**
      - **DB ORM** : JPA
      - **Test** : JUnit + AssertJ

- 대기열 시스템을 구축하고, 예약 서비스는 작업가능한 유저만 수행할 수 있도록 해야합니다.
